
*** yolo 물체 인식 + 챗봇 ***
투명 페트병을 분류해주는 기계 관리 업체 => 업체 웹 사이트
아파트 단지 내 분리수거장에 기계를 설치하여 투명 페트병을 수거한다.
이유 : 사용자들에게 분리수거 장려, 투명 페트병을 활용해 옷이나 가방 등을 제작하는 업체에 팔아서 수익 창출하기

기계 사용법
1. 사용자 번호(아이디,이름 등 신분확인용) 입력
2. 투명 페트병만 투입
3. 갯수에 따라 사용자에게 포인트 제공
	a등급(기본) : 1개당 100포인트
	b등급(페트병 투입 누적 수 100개 이상인 경우) : 1개당 300포인트 
	c등급(페트병 투입 누적 수 500개 이상인 경우) : 1개당 500포인트

위 포인트는 투명 페트병을 재활용하여 만든 상품으로 교환 가능 => 상품 교환 웹 사이트
상품 예시
1000포인트 - 악세서리
5000포인트 - 가방, 신발 등
10000포인트 - 옷


메인페이지
ㄴ 회원가입 => 로그인 페이지로
	ㄴ 이메일(중복불가), 비밀번호, 이름, 연락처

ㄴ로그인 => 메인 페이지로
	ㄴ 이메일, 비밀번호
	ㄴ 이메일 찾기, 비밀번호 찾기

ㄴ마이페이지
	ㄴ 회원 정보 확인
	ㄴ 포인트 확인
	ㄴ 교환 내역 확인
	ㄴ 상품 수령 확인
	ㄴ 회원 정보 수정 => 마이페이지 그대로
	ㄴ 탈퇴 => 메인 페이지로

ㄴ 교환 신청
	ㄴ 사용자 포인트로 교환 가능한 제품 확인 (수량, 상품 체크)
	ㄴ 교환 방법 체크 (현장수령, 배송)
				ㄴ 배송 주소 작성 form

ㄴ 관리자 페이지 => 로그인 회원 등급이 0인 경우 메인 페이지가 아닌 이 페이지로 이동

ㄴ 기계 관리 페이지
- 위치 정보 확인 => 지도 시각화
- 기계 정보 확인 => 최대용량 1000개 중 얼마나 차있는 지 확인 후 => 수거 기능 넣기(수거 버튼 누르면 0으로)

ㄴ 회원 관리 페이지
- 회원 정보 확인 => 탈퇴 시키기 기능
- 교환 신청 관리 => 어떤 회원이 어떤 상품을 교환 신청 했는 지 확인

ㄴ 회원 및 상품 관리 페이지
- 교환 해주는 상품 정보 => 몇 포인트인지, 페트병 몇 개 사용해서 만든 상품인지 등
- 상품 등록, 수정, 삭제
- 위에서 확인한 교환 신청 상품 배송 보내는 페이지
